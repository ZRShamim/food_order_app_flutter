# food_order_app

A Food ordering app

## Code Flow:
For managing state I am using Getx. All the screens are in view folder..

```
└── lib/
    ├── Controllers/
    ├── Models/
    ├── Services/
    └── View/
        └── global_widgets/
        └── screens/
        |   └── widgets/
        └── styles/
```
